JC by console on luogu

前言

在浏览器中，可以使用控制台的 Console 来进行请求。那么，我们能不能利用这一点来快速 JC 别人，甚至隔空 JC 别人呢？答案是，可以的。其实不仅仅是洛谷，其他网站也行，这里只是通过洛谷做一个示范而已。

前置芝士

  1. JavaScript 之 Fetch 操作
  2. JavaScript 之 Promise

洛谷的 API

在洛谷首页，F12打开开发者模式，点“网络”或“NetWork”，然后发一条犇犇，查看请求：

显然，postBenben是发犇犇，watching是获取关注的犇犇，点“postBenben”：

然后点”负载“或”Payloads“：

由此可得，发犇犇是对 /api/feed/postBenben 发送一个POST请求，负载只有一条：”content: 犇犇内容“

其他操作也可以用同样的方法抓包，抓包结果：

操作 地址 参数1名称 参数1值 参数2名称 参数2值
发犇犇 /api/feed/postBenben content 犇犇内容 无 无
发私信 /api/chat/new user 接收方uid content 私信内容
修改个签 /api/user/updateSlogan slogan 新个签 无 无
关注/取关/拉黑 /api/user/updateRelationShip uid 对方uid relationship 新状态（0=取关，1=关注，2=拉黑）

备注：先取关才能拉黑

最后一步

我们能不能把 JC 的代码扔到云剪贴板上呢？当然也可以。直接把代码扔上去（注意一定要原汁原味，不要有任何多余内容），假设云剪贴板编号为xxxxxxxx，那么只要在 Console 中输入 $.get("/paste/xxxxxxxx?_contentOnly",d=>eval(d.currentData.paste.data)) 即可执行其中的代码。
